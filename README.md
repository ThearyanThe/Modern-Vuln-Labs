# Modern-Vuln-Labs

A collection of intentionally vulnerable, Dockerized web applications for practicing advanced penetration testing techniques. Each lab targets a specific vulnerability class and includes a realistic scenario, source code, and Docker setup.

## Quick Start

```bash
git clone https://github.com/your-username/Modern-Vuln-Labs.git
cd Modern-Vuln-Labs
make run-lab LAB=01-union-based-sqli
```

## Lab Structure

- `labs/` – Contains all vulnerability labs, organized by category.
- `scripts/` – Helper scripts for building and cleaning.

## Requirements

- Docker
- Docker Compose
- Make (optional)
